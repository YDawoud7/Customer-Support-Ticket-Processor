# Product Overview

## Acme Workspace
Acme Workspace is our flagship product — an all-in-one productivity platform for creating documents, managing projects, and collaborating with teams.

### Key Features
- **Document Editor**: Rich text editing with real-time collaboration, version history, and templates.
- **Project Boards**: Kanban-style boards for task tracking with customizable workflows.
- **Team Chat**: Built-in messaging with channels, direct messages, and file sharing.
- **File Storage**: Cloud storage with 10GB (Starter), 100GB (Professional), or unlimited (Enterprise).
- **Integrations**: Connect with Slack, Google Workspace, Microsoft 365, GitHub, Jira, and 50+ other tools.
- **Export**: Export documents to PDF, DOCX, PNG, and Markdown formats.

### System Requirements
- **Web**: Chrome 90+, Firefox 88+, Safari 15+, Edge 90+
- **Desktop App**: macOS 12+, Windows 10+, Ubuntu 20.04+
- **Mobile**: iOS 15+, Android 12+
- **Minimum**: 4GB RAM, stable internet connection (5 Mbps recommended)

### Getting Started
1. Sign up at acmecorp.com/signup with email or Google/Microsoft SSO.
2. Choose a plan (Starter, Professional, or Enterprise).
3. Invite team members via email or shareable link.
4. Create your first project and start collaborating.

## Acme Analytics (Add-on)
Available as an add-on for Professional and Enterprise plans. Provides:
- Usage dashboards showing team activity and productivity metrics.
- Custom reports with export to CSV and PDF.
- Goal tracking with automated progress updates.

RESTful API for building custom integrations. See the API Reference documentation for endpoints, authentication, and rate limits.
